Syntax
------

-   [agitcheck2](/agitcheck2 "wikilink")();

Description
-----------

This function will let you check whether the server is currently in WoE:SE mode.

Return Values
-------------

| Value | Description              |
|-------|--------------------------|
| 0     | WoE:SE is currently off. |
| 1     | WoE:SE is currently on.  |
