Syntax
------

-   [strcmp](/strcmp "wikilink")("<string>","<string>")

Description
-----------

This command compares two strings are returns a value:

| Value | Description            |
|-------|------------------------|
| 1     | string 1 &gt; string 2 |
| 0     | strings are equal      |
| -1    | string 1 &lt; string 2 |
