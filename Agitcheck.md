Syntax
------

-   [agitcheck](/agitcheck "wikilink")();

Description
-----------

This function will let you check whether the server is currently in WoE mode.

Return Values
-------------

| Value | Description           |
|-------|-----------------------|
| 0     | WoE is currently off. |
| 1     | WoE is currently on.  |
