Syntax
------

-   **getserverdef**(<type>)

Description
-----------

Returns a server definition. Valid types are listed in :

`VAR_PACKETVER  0`
`VAR_MAX_LEVEL  1`
`VAR_MAX_STORAGE    2`
`VAR_MAX_INVENTORY  3`
`VAR_MAX_ZENY   4`
`VAR_MAX_PARTY  5`
`VAR_MAX_GUILD  6`
`VAR_MAX_GUILDLEVEL 7`
`VAR_MAX_GUILD_STORAGE  8`
`VAR_MAX_BG_MEMBERS 9`
