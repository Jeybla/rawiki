Syntax
------

-   [sqrt](/sqrt "wikilink")(<number>)

Description
-----------

Returns square-root of number.

Note. The returned value is an integer type, so any floating number will be truncated.

Examples
--------

`set .@i, sqrt(25); // .@i will be 5`
`set .@i, sqrt(24); // .@i will be 4`
