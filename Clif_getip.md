Syntax
------

`uint32 `**`clif_getip`**`(void);`

==Parameters==

-   **N/A**

==Description== Gets the Map-Server's IP Address.
==Example==

`uint32 a_ip;`
`a_ip = clif_getip();`

a_ip would have been defined with the server's IP Address.
